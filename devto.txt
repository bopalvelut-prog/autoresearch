I have a 2009 eMachines E727 with a Pentium T4500 that I couldn't throw away. So I made it run AI.

## The toolkit

Three open-source projects that work together:

### 1. Primaclaw — Distributed AI from old hardware

Turns old laptops, Raspberry Pis, and phones into an OpenAI-compatible AI cluster.

- Workers auto-discover via UDP broadcast
- Any OpenAI client can use the cluster (OpenWebUI, SillyTavern, etc.)
- Dashboard for real-time monitoring
- Runs on Alpine Linux and even iPhone (via iSH)

```bash
git clone https://github.com/bopalvelut-prog/e727-local-ai
cd e727-local-ai && pip install -e .
python -m src.coordinator  # on your "brain" machine
python -m src.worker       # on each old machine
```

### 2. AutoResearch CPU Edition — Autonomous AI researcher on any computer

Fork of karpathy/autoresearch. The original needs an H100 GPU. This runs on your laptop.

- AI agent suggests hyperparameter changes via local prima.cpp model
- Runs 5-minute experiments automatically
- Commits improvements, discards failures
- ~12 experiments per hour, ~100 while you sleep

```bash
git clone https://github.com/bopalvelut-prog/autoresearch
cd autoresearch && uv sync
uv run prepare.py
python agent.py  # let it run overnight
```

The agent improved **val_bpb from 2.29 → 2.23** autonomously.

### 3. Model Efficiency Comparator — Find the best model for your hardware

Benchmarks prima.cpp and llama.cpp models by:
- Token speed (tokens/second)
- Intelligence (user-rated quality)
- Model size (smaller = more efficient)
- Security (prompt injection resistance)

```bash
python model_efficiency_comparator.py -p "Explain AI" --format html -o report.html
```

## Real results

| Machine | Year | CPU | Model | Speed |
|---------|------|-----|-------|-------|
| eMachines E727 | 2009 | Pentium T4500 | Qwen2.5 1.5B Q4 | 1.7 t/s |
| iPhone 11 (iSH) | 2019 | A13 (emulated) | Qwen2.5 0.5B | ~0.3 t/s |
| Acer Swift 3 | 2020 | Ryzen 5 4500U | Qwen2.5 3B Q4 | 0.5 t/s |

These are real, measured numbers. Not impressive by cloud standards, but they're running on hardware worth less than a single month of GPT-4 API usage.

## Why?

- **Privacy**: Everything runs locally. No data leaves your network.
- **Cost**: Old hardware is free or nearly free.
- **Learning**: You understand how AI actually works when you see it chugging along at 1.7 tokens per second.
- **Sustainability**: E-waste gets a second life.
